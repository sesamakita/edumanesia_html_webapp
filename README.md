# EduAdmin - Smart School Management System

EduAdmin adalah sistem manajemen sekolah modern yang dirancang untuk memudahkan administrasi dan transparansi data sekolah. Platform ini menyediakan dashboard komprehensif untuk berbagai pemangku kepentingan, mulai dari staf Tata Usaha, Kepala Sekolah, hingga tingkat Kabupaten dan Provinsi.

## 🚀 Fitur Utama

### 📊 Dashboard Multi-Level
- **Admin TU Dashboard**: Manajemen harian sekolah, termasuk absensi, keuangan, dan inventaris.
- **Kepala Sekolah Dashboard**: Monitoring performa sekolah secara keseluruhan.
- **Gubernur & Bupati Dashboard**: Visualisasi data statistik sekolah di tingkat daerah.
- **SuperAdmin Dashboard**: Manajemen sistem tingkat lanjut.

### 🏫 Manajemen Akademik
- **Data Siswa & Guru**: Pengelolaan biodata lengkap civitas akademika.
- **Manajemen Kelas**: Pengaturan rombongan belajar dan jadwal.

### 📂 Administrasi & Keuangan
- **Keuangan / SPP**: Monitoring pembayaran siswa dengan status real-time.
- **Persuratan**: Manajemen surat masuk dan keluar secara digital.
- **Inventaris & Sarpras**: Monitoring aset dan sarana prasarana sekolah.

### ⏱️ Monitoring Real-Time
- **Live Attendance**: Tracking absensi siswa dan guru menggunakan sistem QR Scan.
- **Statistik Dinamis**: Visualisasi data menggunakan grafik yang interaktif.

## 🛠️ Tumpukan Teknologi
- **Frontend**: HTML5, Vanilla CSS3 (Custom Glassmorphism UI)
- **Logika**: JavaScript (ES6+)
- **Grafik**: [Chart.js](https://www.chartjs.org/)
- **Ikon**: [Google Material Icons](https://fonts.google.com/icons)
- **Topografi**: Inter & Poppins (via Google Fonts)

## 💻 Cara Penggunaan
Sistem ini dirancang sebagai aplikasi web berbasis HTML statis untuk kebutuhan prototyping dan dashboard frontend.

1. Clone repositori ini ke komputer lokal Anda.
2. Buka file `index.html` menggunakan browser modern (Chrome, Edge, atau Firefox).
3. Anda akan diarahkan ke Dashboard utama secara otomatis untuk keperluan demonstrasi.
